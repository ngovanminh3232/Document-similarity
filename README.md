# Document-similarity
## Tổng quan dự án
Dự án Document similarity nhằm thực hiện mục đích là so sánh các văn bản.

## Ứng dụng của dự án
- So sánh các văn bản đưa ra gợi ý.
- So sánh các các bài báo giống nhau đưa ra gợi ý cho người dùng.
